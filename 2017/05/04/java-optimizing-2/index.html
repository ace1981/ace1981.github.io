    • Java堆 存放所有的java对象 逃逸分析Escape Analysis 相关JVM参数

      ​ -XX:+PrintEscapeAnalysis 开启逃逸分析后，可通过此参数查看分析结果。
      ​ -XX:+EliminateAllocations 开启标量替换
      ​ -XX:+EliminateLocks 开启同步消除
      ​ -XX:+PrintEliminateAllocations 开启标量替换后，查看标量替换情况。
