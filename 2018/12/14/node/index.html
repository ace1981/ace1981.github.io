nvm,node,npm之间的区别

  • nvm

    nodejs版本管理工具。

  • nodejs

    在项目开发时的所需要的代码库

  • npm

    在安装的nodejs的时候，npm也会跟着一起安装，它是包管理工具。

    npm管理nodejs中的第三方插件

常用命令
安装hexo

npm install -g hexo-cli

重建node-sass

npm rebuild node-sass
